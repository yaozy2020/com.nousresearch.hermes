快速向导

4 步完成首次配置
本向导只完成最小化配置（安装 + 选 Provider + 启动服务）。API Key、Token、模型选择请在第 4 步打开官方 Dashboard 配置。

第 1 步 · 检查 Hermes 安装

本步骤会检查 Python 包 hermes-agent 是否已安装到本地 venv。如未安装，点「一键安装」自动从 PyPI 拉取（需 1-3 分钟）。

检查中…

第 2 步 · 选择 Provider 预设

选一个 LLM 服务商。点选后下方可修改 Base URL（OpenAI 兼容/Ollama 等）并可选填入 API Key，留空则到 Dashboard 再填。

第 3 步 · 启动 Gateway

Gateway 是消息通道服务，负责转发 Telegram/Discord/Slack 等渠道的消息到 Hermes Agent。本机以 systemd 用户进程方式运行，无端口暴露。

准备中…

第 4 步 · 启动 Dashboard

Dashboard 是 Hermes 官方控制台（端口 9119），在这里填入 OpenAI/OpenRouter 等 API Key、配置 Telegram Bot Token、绑定渠道。点下方按钮启动后再点「打开 Dashboard」跳转。

准备中…

消息频道

配置 Hermes Gateway 的消息接入渠道
填写后保存即写入 .env，需重启 Gateway 生效。复杂频道（OAuth/外部服务）请打开 Dashboard 配置。

🟢 直接配置（重启 Gateway 生效）

填写下方表单 → 保存 → 重启 Gateway → 频道立即可用。

🟡 进阶频道（需配置文件 + 引导）

这些频道需修改 config.yaml 或经过 OAuth/回调，建议打开 Dashboard 配置。

DingTalk 钉钉
需在钉钉开放平台注册应用，配置 OAuth 回调 URL。
Feishu 飞书
需企业自建应用 + 事件订阅回调。
Email 邮件
SMTP/IMAP 多字段配置。
Webhook
需在 config.yaml 定义路由及 HMAC 密钥。

🔴 外部依赖（不可在面板中配置）

需先部署外部服务或申请商业账号，配置完成后请打开 Dashboard 接入。

WhatsApp
需 WhatsApp Business 账号。

需 macOS 服务器。
SMS
需 Twilio 账号。
Matrix
需 Matrix Homeserver + token。
Yuanbao 元宝
腾讯元宝（需手机号）。
Weixin 微信公众号

WeCom Callback
